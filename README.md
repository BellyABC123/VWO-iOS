# VWO for iOS (beta)
This is the beta release of VWO A/B Testing tool for iOS apps.

## Requirements
Build Target must be iOS 5.1 or above.

## Getting Started
To install VWO on your iOS app, please refer to our **[Quick Start Guide](http://vwo.com/ios)**.
